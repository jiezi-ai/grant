# 诚信规则

## I1. 禁止购买 Star

第二阶段要求的 50 star 必须来自真实用户。项目方会抽查 stargazer 列表，若发现大量空号或新注册账号，视为违规。

## I2. 禁止重复申请

每人只能申请一次第一阶段（见 [eligibility.md](eligibility.md) E4）。使用多个 GitHub 账号重复申请将被取消资格。

## I3. 禁止转售资源

获得的 Coding Plan、API Token、服务器资源仅供申请人本人使用，不得转售或转让。

## I4. 学生身份真实性

申请人必须确保提供的学校、专业、年级信息真实。如发现虚假信息，将取消资助资格。

## I5. 违规处理

违反以上规则的申请人：
- 立即取消当前阶段资助
- 不得申请后续阶段
- 在 `records/` 中记录处理结果
