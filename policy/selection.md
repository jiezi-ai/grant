# 筛选规则

## S1. 第一阶段入选

- 方式：**先到先得**
- 申请人在 jiezi.ai/apply 提交信息后，到 GitHub 提交 Issue 触发审核
- 审核通过 + edu 邮箱验证 = 有效申请
- 每批满 100 人后截止

## S2. 第二阶段筛选

- 筛选比例：约 **10:1**（600 人中约 60 人）
- 申请窗口：第一阶段结束后 **两个月内**
- 滚动申请，随时提交
- 申请资格：
  1. 已完成第一阶段
  2. 在 [jiezi-ai](https://github.com/jiezi-ai) 组织下有一个 GitHub repo
  3. 该 repo 由 **AI agent 辅助创建**
  4. 该 repo 至少 **50 star**（申请资格线）
- 成果形式不限：产品、工作流、AI 使用方式创新
- 评审方式：达到申请资格后，由发起人**人工 review**，综合评估项目的真实价值、完成度和创新性
- 50 star 是提交申请的最低门槛，不代表达到即通过

## S3. 第三阶段选拔

- 人数：**3-5 人**
- 从第二阶段优秀学生中选拔
- 由发起人综合评估，考量维度：
  - 成果的真实价值
  - 持续学习和成长
  - 长期投入意愿

## S4. 附加机会

从第二阶段开始提供：
- 实习撮合（对接赞助企业）
- 项目展示（社交媒体曝光）
- 导师辅导（与发起人一对一交流）
