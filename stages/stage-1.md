# 第一阶段：体验期

> 让每一个想接触 AI 的大学生，都有机会亲手体验。

## 你会得到

**AI Coding Plan** — 1 个月

支持主流编程工具（Claude Code、Cline、Roo Code 等），让你用 AI 辅助编程、构建项目。

> 当前提供：MiniMax Plus 极速版（M2.7-highspeed，100 TPS 极速推理）。具体供应商可能根据市场情况调整，以确保最优性价比。

## 名额

- 每批 **100 人**
- 每两个月开放一批
- 全年共 **6 批，600 人**
- 先到先得，满员即止

## 申请条件

1. **在校大学生**（本科/硕士/博士，不限专业）
2. **Star** 本仓库
3. **Fork** 本仓库，在 `students/` 目录下创建以你 GitHub 用户名命名的 `.md` 文件，按模板填写信息
4. **提交 PR**
5. 通过 edu 邮箱验证后，加微信完成确认

详细操作步骤请看 [申请指南](../docs/apply-guide.md)。

## 说明

- 这是**无条件赠送**，不要求你提交成果
- 但如果你想进入[第二阶段](stage-2.md)获得更多资源，你需要在体验期结束后两个月内拿出成果
- 我们鼓励你用 AI 去做任何你感兴趣的事
