# 第一阶段：体验期

> 让每一个想接触 AI 的大学生，都有机会亲手体验。

## 你会得到

**API Token** — $20 额度，有效期 1 个月

支持主流编程工具（Claude Code、Cline、Roo Code 等），让你用 AI 辅助编程、构建项目。

> 当前提供：$20 API Token，支持 20+ 主流模型（Claude、Gemini、GPT、DeepSeek、MiniMax、GLM 等）。验证通过后自动发放到邮箱。额度有效期 1 个月，到期未用完的部分自动回收。

## 名额

- 每批 **100 人**
- 每两个月开放一批
- 全年共 **6 批，600 人**
- 先到先得，满员即止

## 申请条件

1. **在校大学生**（本科/硕士/博士，不限专业）
2. 在 [jiezi.ai/apply](https://jiezi.ai/apply) 填写信息，获取申请码
3. 到本仓库 [提交 Issue](https://github.com/jiezi-ai/grant/issues/new?template=apply.yml)，填入申请码
4. 通过 edu 邮箱验证后，API Token 自动发放到邮箱

个人信息通过网站提交，不会出现在公开的 Issue 中。详细操作步骤请看 [申请指南](../docs/apply-guide.md)。

## 说明

- 这是**无条件赠送**，不要求你提交成果
- 但如果你想进入[第二阶段](stage-2.md)获得更多资源，你需要在体验期结束后两个月内拿出成果
- 我们鼓励你用 AI 去做任何你感兴趣的事
