# 第二阶段：深度资助

> 第一阶段让你体验 AI，第二阶段让你用 AI 做出东西。

## 你会得到

| 资源 | 时长 |
|------|------|
| AI Coding Plan | 3 个月 |
| 云服务器 | 1 年 |
| API Token（$50 额度） | 一次性发放 |

> 当前 Coding Plan 供应商：MiniMax；API Token 供应商：OpenRouter。具体供应商可能调整。

## 名额

- 约 **60 人**（从第一阶段 600 人中筛选，约 10:1）
- **滚动申请**，随时提交

## 申请条件

1. **完成过第一阶段**
2. 在 [jiezi-ai](https://github.com/jiezi-ai) 组织下有一个 GitHub repo
3. 该 repo 由 **AI agent 辅助创建**
4. 该 repo 至少 **50 star**
5. 成果形式不限：产品、工作流、AI 使用方式创新，都可以
6. **申请窗口**：第一阶段结束后 **两个月内**

## 组队

可以组队，也可以单干。有人擅长独立作战，有人擅长协作分工，两种都好。

## 附加机会

从第二阶段开始，我们会为学生提供：

- **实习撮合** — 对接赞助企业的实习机会
- **项目展示** — 在社交媒体上展示你的成果
- **导师辅导** — 与发起人一对一交流

## 之后

如果你在第二阶段持续投入、持续成长，你有机会进入[第三阶段](stage-3.md)——获得一整年的长期资助。
